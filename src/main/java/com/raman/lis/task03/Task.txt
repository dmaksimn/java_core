Необходимо реализовать интерфейс а затем реализовать его в классе завод.
Класс завод должен быть абстрактным.
От него должны наследоваться все остальные заводы.
Общее колличество заводов на ваше усмотрение, минимум 3.
На заводе возможно производить только автомобили которые допустимы на данном заводе.


Класс Car должен быть астрактным.
Для каждого завода должна быть своя реализация автомобиля который он производит,
можно несколько.

Сервис должен быть абстрактным. И реализовывать интерфейс который будет задавать его поведение.
Для каждой задачи необходим конкретный класс сервиса.
Например сервис для покраски, или сервис для замены колёс.

Салон должен изменять автомобиль на необходимом сервисе.
Так же заказ автомобиля должен происходить с нужного завода.

Ключевые моменты:
1) Форматирование
2) Исключение дубликатов кода.
3) Исправление/завершение прошлого этапа.
